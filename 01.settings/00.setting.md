# 챗봇 개발 환경 설정

## 공통 사전 준비
1. Python 설치 (3.13 권장)
2. UV 설치

---

## Windows

```powershell
# 1) 가상환경 생성 (Python 3.13)
uv venv --python 3.13

# 2) 파이썬 버전 확인
.venv\Scripts\python --version

# 3) 가상환경 활성화
.venv\Scripts\activate

# 4) 가상환경 종료 (필요할 때만)
deactivate

# 5) 라이브러리 설치 (requirements.txt 한 번에 설치)
uv pip install -r requirements.txt
```

---

## macOS

```bash
# 1) 가상환경 생성 (Python 3.13)
uv venv --python 3.13

# 2) 파이썬 버전 확인
.venv/bin/python --version

# 3) 가상환경 활성화
source .venv/bin/activate

# 4) 가상환경 종료 (필요할 때만)
deactivate

# 5) 라이브러리 설치 (requirements.txt 한 번에 설치)
uv pip install -r requirements.txt
```
