Python 설치
UV설치
uv venv --python 3.13 파이선 버전 설치
.venv/bin/python --version 파이선 버전확인
source .venv/bin/activate 가상환경 생성 / deactivate 가상황경 종료
uv pip install python-dotenv 환경 변수 관리 라이브러리
uv pip install -q -U google-genai Google GenAI SDK 설치
uv pip install google-generativeai 구글에서 공식으로 배포한 Generative AI 파이썬 SDK
